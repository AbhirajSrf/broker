# Buyer Portal for Real Estate

## How to Run

1. **Unzip and extract** the project folder to a location of your choice.

2. **Open the project in VS Code** and open **two terminals**.

3. In the **first terminal**, navigate to the backend folder:

   ```bash
   cd backend
   ```

4. In the **second terminal**, navigate to the frontend folder:

   ```bash
   cd frontend
   ```

5. **In both terminals**, run the following to install all dependencies:

   ```bash
   npm install
   ```

6. **In both terminals**, start the development server:

   ```bash
   npm run dev
   ```

7. In the terminal running the frontend, **hold `Ctrl` and click** the local address shown (e.g. `http://localhost:5173`) to open the app in your browser.
